HDRF’s block mirroring Finished Professor Kazuhiko Kato (University of Tsukuba), Professor Hirotake Abe (University of Tsukuba) One of the main issues with HDRF is that any communications between the nodes are using the original data with its original size, wasting network bandwidth and compute because the data will be recompressed again at the receiving node. Because of the modularity of HDRF, we can solve this issue through an additional module called block mirroring. Block mirroring replaces the block replication function of HDFS and transfer the reduced data instead of the original data, minimizing the network usage and compute. https://github.com/NSWRyan/HDRF
Data reduction schemes in Distributed File Systems Finished Professor Kazuhiko Kato (University of Tsukuba), Professor Hirotake Abe (University of Tsukuba) A project that evaluates the effectivity of data reduction schemes like compressions and deduplications in distributed file systems. The project includes a proposal for including data reduction schemes directly in the DFS layer. Please check the APSys2020 paper for more details. https://github.com/NSWRyan/HDRF
High throughput Content-Dependent Chunking (CDC) algorithm Finished Professor Hyotaek Lim (Dongseo University), Dr. Mohammed Atiquzzaman (University of Oklahoma) An algorithm that produces variable sized offsets based on the file content. The algorithm is aimed for CDC algorithm used in deduplication applications.
Smart Deduplication Mobile Finished Professor Hyotaek Lim (Dongseo University), Dr. Mohammed Atiquzzaman (University of Oklahoma) An Android application aimed toward mobile cloud applications. The proposed application uses deduplication to minimize the storage consumption and data transfer between the user and cloud. It includes an adaptive learning system to select between file deduplication and variable sized-block deduplication depending on the file types.
An Andorid Application for controlling Quadcopter Finished Petrus Santoso (Petra Christian University) An undergraduate project that involved reverse engineering Parrot AR.Drone 2.0 packets’ content and reimplementing it in an Android application. Function includes drone control, status display, and video image display. Please check the project code in GitHub.
