3D Led cube

Status: Postponed

Description :

An Arduino controlled project consisting of 8x8x8 (row * column * level) LEDs in a cube shape (like this). The goal is to create a 3D display by using the LEDs in cube geometric. The key challenge in the project is to expand the number of the output of Arduino to control 512 LEDs, which is impossible with Arduino alone. To achieve this, I engineered a solution that used demultiplexers, which is to select row and level and latch, which is to store 8 bits. With the combinations of scanning techniques (persistence of vision (POV)), latch, and demultiplexers, we can expand the limited output of Arduino to control more than 500 LEDs with a single Arduino!

However, the project failed because my electricity set up in my house was not grounded properly and I killed most of the LEDs during soldering through electrostatic discharge (ESD). I learned this reason when I worked as an EMC engineer a year later.

OpenCV pencil touch

Status: Completed (somehow).

Description:

This is a project that uses OpenCV, infrared camera (like Wiimote camera or CCTV camera), and printed circles to control your computer.

Usage instruction:

  1. First, we attach the printed circles to each corner of a monitor (4 if you have a normal monitor).
  2. Aim the camera at the monitor screen.
  3. Attach a printed circle to a pencil.
  4. Run the program and use the pencil as a mouse to control the pointer.

How does it work?: The magic behind it is OpenCV image processing. The OpenCV code processes the images from the camera, find the circles in the camera, and finally return the coordinates of those circles to a program. With these coordinates, we can know the position of the pencil relative to the monitor screen through some mathematic formulas. Finally, the program passed the coordinate as a mouse movement such that the user can control the computer using the pencil.

The code is written in combination of C++ and C#. I will try to find the code and upload it to Github if I can find it.

Mic amplifier

Status: Completed.

Description: The world is in crisis and most of the meetings occur online. However, good microphones cost a lot. This project uses existing stuff that I have to create a useable microphone to use in online meetings.

Ingredients:

  1. Electret microphone (I used one out of an unused phone mic/headphone combo).
  2. MAX9814 amplifier module (my Chinese friend got it for me from China for around $2 like this, thanks Haku-san).
  3. DAC (I used Behringer UCA202, but you can use your onboard DAC in your computer for that).

Instruction:

  1. Connect the microphone and 5V supply to the MAX9814 module.
  2. Connect the module to the DAC.
  3. Adjust the gain to your liking.
  4. Done!

How does it work?:

MAX9814 is an auto gain controller (AGC) for electret microphones (like those in your laptops and phones). It increases or decreases sound (or normalize it) such that your voice is not too small to hear. One of the drawbacks is that the 5V supply must be clear of ripples. Using 5V from the computer might not be a good idea because the noise from the computer will also affect the output of the amplifier. I found that power banks and iPad chargers work very well without any noise with the amplifier.
