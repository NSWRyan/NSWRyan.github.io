3D LED　キューブ

状態: 延期

説明 :

An Arduino controlled project consisting of 8x8x8 (row * column * level) LEDs in a cube shape (like this). The goal is to create a 3D display by using the LEDs in cube geometric. The key challenge in the project is to expand the number of the output of Arduino to control 512 LEDs, which is impossible with Arduino alone. To achieve this, I engineered a solution that used demultiplexers, which is to select row and level and latch, which is to store 8 bits. With the combinations of scanning techniques (persistence of vision (POV)), latch, and demultiplexers, we can expand the limited output of Arduino to control more than 500 LEDs with a single Arduino!

However, the project failed because my electricity set up in my house was not grounded properly and I killed most of the LEDs during soldering through electrostatic discharge (ESD). I learned this reason when I worked as an EMC engineer a year later.

OpenCVタッチ鉛筆

状態: 完成

説明:

Have you ever wish to use your pencil and use it to control your computer? Then, this project might be for you! This is a project that uses OpenCV, infrared camera (like Wiimote camera or CCTV camera), and printed circles. First, we attach the printed circles to each corner of a monitor (4 if you have a normal monitor). The camera is then aimed at a monitor screen and the user can use a pencil with a circle attached to it as a pointer to use as a mouse-like stylus. The magic behind it is OpenCV image processing. The OpenCV code processes the images from the camera, find the circles in the camera, and finally return the coordinates of those circles to a program. With these coordinates, we can know the position of the pencil relative to the monitor screen through some mathematic formulas. Finally, the program passed the coordinate as a mouse movement such that the user can control the computer using the pencil.

The code is written in combination of C++ and C#. I will try to find the code and upload it to Github if I can find it.

マイクアンプ

状態: 完成

説明: コロナのせい、今の会議やゼミはオンラインになった。そのため、音量がいいマイクが必要になった。このプロジェクトは私が古いマイクを使い、最低限の予算で古いマイクをオンライン会議で使えるように改善する事業だである。

素材：

  1. エレクトレットマイク (ヘッドホンで使われたマイク).
  2. MAX9814 アンプモジュール（中国から来た友達に頼まれ、中国で買ってもらった。MAX9814, 白さんありがとう！）。
  3. DAC　（サウンドカード）、このプロジェクトはBehringer UCA202を使用したが、内蔵サウンドカードも使える。

作る方法:

  1. マイクと５V電源をマイクアンプにつなぐ。いい電源は重要だ。電源が静的に悪いの場合、音質も悪くなる。iPadの電源がお勧めする。
  2. アンプモジュールとDACに接続する。
  3. アンプモジュールがお好みに調整する.
  4. 完成！

なぜいい音量出られる？

MAX9814 は自動音量調整が出来るマイクアンプである。しかし、マイクはエレクトレットマイクにかぎる。そのアンプは自動でマイクから音量を調整できる。音量が低い場合、音量が上がる。逆に音が大きすぎると、音量が下がる。しかし、電源の品質は重要である。なぜなら、電源が純粋ではない場合、音質は悪くなる。モバイルバッテリーやiPadの電源はお勧めである。

このプロジェクトのように、私のプロジェクトはほぼ私が不便を感じた問題を基づき、問題を解決プロジェクトである。ほかのプロジェクトが覚えられたらもっと入れておる。
