About Me

Hello world! This is a resume page of Ryan Nathanael Soenjoto Widodo. In 1991, Widodo was born in a loving Indonesian Christian family of 4 persons. Widodo is currently pursuing PhD degree at University of Tsukuba under the guidance of Professor Kazuhiko Kato and Professor Hirotake Abe in Operating System and System Software (OSSS) Laboratory Operating System and System Software (OSSS) Laboratory.

Research Interest

Widodo has various research interest in Computer Science, including:

  1. Key-value storage
  2. Persistent memory
  3. Big data
  4. Deduplication and compression
  5. Mobile cloud computing

Projects

Name Status Comments
An Andorid Application for controlling Quadcopter Finished An undergraduate project that involved reverse engineering Parrot AR.Drone 2.0 packets' content and reimplementing it in an Android application. Function includes drone control, status display, and video image display. Please check the project code in GitHub.
Smart Deduplication Mobile Finished An Android application aimed toward mobile cloud applications. The proposed application uses deduplication to minimize the storage consumption and data transfer betweeen the user and cloud. It includes an adaptive learning system to select between file deduplication and variable sized-block deduplication depending on the file types.
High throughput Content-Dependent Chunking (CDC) algorithm Finished An algorithm that produces variable sized offsets based on the file content. The algorithm is aimed for CDC algorithm used in deduplication applications.
Data reduction schemes in Distributed File Systems Finished A project that evaluates the effectivity of data reduction schemes like compressions and deduplications in distributed file systems. The project includes a proposal for including data reduction schemes directly in the DFS layer. Please check the APSys2020 paper for more details.
